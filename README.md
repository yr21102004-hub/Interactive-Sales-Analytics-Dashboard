# 📊 Interactive Sales Analytics Dashboard

An interactive data analytics dashboard built using Python and Streamlit.  
This project demonstrates real-world data cleaning, analysis, and visualization techniques on a sales dataset.

---

## 🚀 Features

- 🧹 Data Cleaning
  - Handling missing values
  - Removing duplicate rows
  - Converting date from string to datetime

- 🎛️ Interactive Filters
  - Filter data by City and Category

- 📊 Data Visualization
  - Sales by City
  - Sales by Category
  - Sales over time
  - Distribution of sales

- 📈 Key Metrics (KPIs)
  - Total Sales
  - Total Orders
  - Average Sales

- 🧠 Insights Generation
  - Identify top-performing cities
  - Detect best-selling categories

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn

---

## 📁 Dataset

The dataset used in this project is synthetically generated and includes:

- Missing values
- Duplicate rows
- Date stored as string
- Multiple categories and cities
